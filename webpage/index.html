Pilas, un motor para hacer juegos

Introducción

Pilas te permite hacer tus primeros videojuegos de manera sencilla, ya que ha sido diseñado para estudiantes y jovenes.

Está construido sobre el lenguaje de programación python y la biblioteca SFML.

Entre sus características vas a encontrar:

  • Enteramente en español, tanto el código como la documentación.
  • Utiliza aceleración de video para rotar, escalar y realizar efectos.
  • Incluye personajes ya diseñados listos para usar.
  • Es software libre y gratis.
