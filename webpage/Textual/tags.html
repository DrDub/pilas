The outer container may have these values (indicating width, following the underscore; please see the source for the sample layouts page for examples): the numbers from 5 to 100, in increments of 5, indicating a percentile width; resolution-specific pixel widths of 600px, 760px, 980px; and width in ems from 5em to 100em, in increments of 5.

Each column should use an .inner div tag inside of it (just like tires need inner tubes). In order to give it borders and a background-color, add the class box to it. A column can use the values (indicating width) from 5% to 100%, in increments of 5.

For classes applied to body elements, there are three for text-alignment: text_center, text_left, and text_right. For making stuff embed itself (such as photos) there is float_left and float_right. In oder to ensure that a given tag appears below the matching type of floating element, just add the clear_left, clear_right or clear_both.
